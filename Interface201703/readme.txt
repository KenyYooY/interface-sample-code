概要
================
　このアーカイブには，表題の記事の中で作成したプログラム・ファイルやデータ・ファイルが収録されています．
　詳しくは該当記事を参照してください．
　

================

================
　プログラムは筆者の元で動作を確認済みです．

=================
アーカイブの内容
=================
￥
├-data
│  ├faces-positive - 認識したい顔画像を入れてください。
│  │     └negative - positiveと異なる顔画像を入れてください。
│  │
│  └lfw - 「Labeled Faces in the Wild」の画像データを入れてください。
│
├-extract_face.py - 顔領域を抽出するコードです。
│
├-model.py - ニューラルネットワークのモデルを構築するコードです。
│
├-train.py - ニューラルネットワークを学習するコードです。
│
├-detect_video_face.py - 学習したニューラルネットワークを使って顔を検出します。
│
├-util.py - 共通で利用する関数を記述しています。
│
└─readme.txt
